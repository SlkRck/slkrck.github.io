# Azure Snippets (SNIPS)

This directory contains reusable Azure-related PowerShell snippets intended to be imported, dot-sourced, or embedded into larger scripts and modules.

## Structure

- **functions/**  
  Individual reusable PowerShell functions.

- **modules/**  
  Module-oriented Azure components or scaffolding.

These snippets are not standalone scripts and are meant to be composed into higher-level automation.
