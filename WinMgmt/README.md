# Windows Management (WinMgmt)

This directory contains PowerShell scripts, modules, snippets, and tests for Windows system management and automation.

The content here focuses on:
- Remote management
- WinRM and CIM usage
- System inspection and diagnostics
- Consistent scripting patterns

## Structure

- **MODULES/**  
  PowerShell modules for Windows management.

- **SCRIPTS/**  
  Executable scripts and interactive tools.

- **SNIPS/**  
  Reusable helper functions and fragments.

- **PesterTests/**  
  Tests validating WinMgmt behaviors.
