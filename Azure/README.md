# Azure Automation

This directory contains Azure-focused automation, scripts, snippets, and tests.

The content here is intended to support inventory, inspection, and operational workflows across Azure environments.

## Structure

- **SCRIPTS/**  
  Executable Azure automation scripts.

- **SNIPS/**  
  Reusable functions and module fragments for Azure-related tasks.

- **PesterTests/**  
  Tests validating Azure scripts and behaviors.

## Notes

Scripts and snippets in this directory assume appropriate Azure authentication and permissions are already in place.
Refer to individual script headers and module documentation for specific usage details.
