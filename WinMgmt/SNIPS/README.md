# WinMgmt Snippets (SNIPS)

This directory contains reusable Windows management snippets.

These are not standalone scripts and should be imported or dot-sourced into higher-level tooling.

## Structure

- **functions/**  
  Reusable PowerShell functions.

- **modules/**  
  Module-oriented components and scaffolding.
  Each snippet or module should include its own documentation where applicable.
